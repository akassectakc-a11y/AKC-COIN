## 📚 문서 개요

AKASSECT (AKC)의 공식 한글 백서입니다. AI 기반 블록체인 결제 인프라에 대한 전체적인 비전과 기술적 세부사항을 제공합니다.

---

## 🌐 웹 버전 보기

### 방법 1: 로컬 웹 서버

```bash
# 현재 디렉토리에서
python3 -m http.server 8080
```

그 다음 브라우저에서:
```
http://localhost:8080/whitepaper.html
```

### 방법 2: Node.js 서버

```bash
# 현재 디렉토리에서
npx http-server -p 8080
```

### 방법 3: VS Code Live Server

1. VS Code에서 `whitepaper.html` 파일 열기
2. 우클릭 → "Open with Live Server"
