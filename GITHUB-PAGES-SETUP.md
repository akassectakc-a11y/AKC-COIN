# 🌐 GitHub Pages 무료 웹사이트 설정

## 📋 **GitHub Pages 활성화 방법**

### **1. GitHub 리포지토리에서 설정**
1. GitHub 리포지토리 페이지로 이동
2. **Settings** 탭 클릭
3. 왼쪽 메뉴에서 **Pages** 클릭
4. **Source**에서 **Deploy from a branch** 선택
5. **Branch**에서 **main** 선택
6. **Save** 클릭

### **2. 자동 생성되는 URL**
```
https://akassectakc-a11y.github.io/AKC-COIN/
```

### **3. 사용 가능한 페이지들**
```
메인 페이지: https://akassectakc-a11y.github.io/AKC-COIN/add-token-button.html
토큰 정보: https://akassectakc-a11y.github.io/AKC-COIN/CONTRACT-ADDRESS.md
사용자 가이드: https://akassectakc-a11y.github.io/AKC-COIN/ADD-TO-METAMASK.md
```

## ✅ **장점**
- **무료**: GitHub에서 제공
- **자동 업데이트**: Git 푸시 시 자동 반영
- **SSL**: HTTPS 자동 제공
- **CDN**: 전 세계 빠른 접속

## 🎯 **현실적 조언**

### **웹서버 불필요한 경우:**
- 토큰만 사용하는 경우
- 소규모 커뮤니티
- 비용 절약 필요

### **GitHub Pages 추천하는 경우:**
- 사용자 편의성 중요
- 전문적인 이미지 필요
- 링크 공유 편의성

## 💡 **결론**

**웹서버는 선택사항입니다!**

- **토큰 기능**: 웹서버 없이도 완벽 작동 ✅
- **사용자 접근**: 지갑에서 직접 추가 가능 ✅
- **정보 공유**: GitHub + BscScan으로 충분 ✅
