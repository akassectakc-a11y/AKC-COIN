# 🥞 PancakeSwap 유동성 풀 생성 가이드

## 🎯 **목표**
AKC/BNB 유동성 풀을 생성하여 토큰 거래를 가능하게 하고 자동 로고 인식을 활성화합니다.

## 📋 **준비사항**

### **필요한 자산**
- **AKC 토큰**: 풀에 추가할 수량
- **BNB**: 페어링할 BNB 수량
- **가스비**: 트랜잭션 수수료용 BNB

### **권장 유동성**
- **최소**: $1,000 상당 (AKC + BNB)
- **권장**: $10,000 상당 (더 안정적인 가격)
- **이상적**: $50,000+ (거래소 등록에 유리)

## 🔧 **단계별 가이드**

### **1단계: PancakeSwap 접속**
```
URL: https://pancakeswap.finance/liquidity
```

### **2단계: 지갑 연결**
- MetaMask 연결
- BSC 메인넷 확인
- 충분한 BNB 가스비 확보

### **3단계: 토큰 추가**
```
토큰 주소 입력: 0x02D8b729885290a3CA724F3Df5793b74Ff226A17
토큰 심볼 확인: AKC
토큰명 확인: AKASSECT
```

### **4단계: 유동성 비율 설정**
```
예시 (1 BNB = $600 기준):
- 1 BNB + 해당하는 AKC 수량
- 초기 가격 설정 (예: 1 AKC = $0.002)
- 총 300,000 AKC = $600 (1 BNB와 동일)
```

### **5단계: 유동성 추가 실행**
- 수량 확인
- 슬리피지 설정 (0.5-1%)
- 트랜잭션 승인
- 완료 확인

## ⚠️ **주의사항**

### **임팩트 손실**

- 급격한 철회 시 가격 변동성 증가

### **가격 추적**
```javascript
// 현재 토큰 가격 확인

  ethers.parseEther("1"), // 1 AKC

);
console.log("1 AKC =", ethers.formatEther(price[1]), "BNB");
```
