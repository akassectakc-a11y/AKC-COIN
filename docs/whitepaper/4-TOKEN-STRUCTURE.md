# 4. AKC 코인 구조 (AKC Token Structure)

## 4.1 토큰 기본 정보
| 항목 | 내용 |
|------|------|
| 토큰명 | AKC (AI-K Coin) |
| 표준 | BEP-20 |
| 총 발행량 | 300,000,000 |
| 네트워크 | Binance Smart Chain |
| 컨트랙트 주소 | 0x02D8b729885290a3CA724F3Df5793b74Ff226A17 |

## 4.2 토큰 분배
| 구분 | 비율 | 용도 |
|------|------|------|
| 보안 예치(결제 보증) | 30% | AI 보증형 결제 리저브 |
| 커뮤니티 리워드 | 25% | 거래 참여 보상 |
| 파트너 및 연동사 | 20% | 결제 시스템 제휴 |
| 개발 및 운영 | 15% | 시스템 유지보수 |
| 예비금 | 10% | 비상 유동성 확보 |

## 4.3 코인 활용성
- P2P 거래 결제수단  
- 스마트 에스크로 예치금  
- 거래 리워드 / 수수료 감면  
- AI 위험분석 서비스 이용료
