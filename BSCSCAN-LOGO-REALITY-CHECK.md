## 🔧 **BscScan 로고 등록 과정**

### **1단계: 컨트랙트 검증 (필수)**
```bash
# BscScan API 키 필요
npx hardhat verify --network bscMainnet 0x02D8b729885290a3CA724F3Df5793b74Ff226A17 "0xb6fbdb9af4c956ca953c4536fec5b28361cadac1"
```

**문제**: API 키가 필요하거나 설정 오류 가능

### **2단계: 로고 제출**
1. **BscScan 고객센터**: https://bscscan.com/contactus
2. **제목**: "Token Logo Submission Request"
3. **첨부**: 256x256 PNG 로고 파일
4. **정보**: 컨트랙트 주소, 토큰명, 심볼

### **3단계: 승인 대기**
- **검토 기간**: 1-4주
- **조건**: 컨트랙트 검증 완료 필수
- **결과**: 승인 시 로고 자동 표시
