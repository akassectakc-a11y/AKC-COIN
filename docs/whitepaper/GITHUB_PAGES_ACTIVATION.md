# 🚀 AKASSECT 백서 GitHub Pages 활성화 가이드

- **완성 주소**: `https://akassectakc-a11y.github.io/AKC-COIN/docs/whitepaper/`

## 📋 **GitHub Pages 활성화 단계**

### **1단계: GitHub 저장소 접속**
1. `https://github.com/akassectakc-a11y/AKC-COIN` 접속
2. 저장소 관리자 권한으로 로그인

### **2단계: Settings 메뉴 접근**
1. 저장소 상단 메뉴에서 **"Settings"** 클릭
2. 왼쪽 사이드바에서 **"Pages"** 클릭

### **3단계: 소스 설정**
1. **Source** 섹션에서:
   - **"Deploy from a branch"** 선택
2. **Branch** 섹션에서:
   - **Branch**: `main` (또는 `master`) 선택
   - **Folder**: `/docs/whitepaper` 선택
3. **"Save"** 버튼 클릭

### **4단계: 배포 확인**
1. 설정 저장 후 약 **5-10분** 대기
2. Pages 설정 페이지에서 **"Your site is published at"** 메시지 확인
3. 생성된 주소: `https://akassectakc-a11y.github.io/AKC-COIN/docs/whitepaper/`

## 🔧 **추가 설정 (선택사항)**

### **✅ 커스텀 도메인 설정**
1. **Pages** 설정에서 **"Custom domain"** 섹션
2. 원하는 도메인 입력 (예: `whitepaper.akassect.com`)
3. DNS 설정에서 CNAME 레코드 추가:
   ```
   whitepaper.akassect.com → akassectakc-a11y.github.io
   ```

### **✅ HTTPS 강제 설정**
1. **Pages** 설정에서 **"Enforce HTTPS"** 체크박스 활성화
2. 모든 HTTP 요청이 HTTPS로 자동 리다이렉트

## 🚨 **문제 해결**

### **❌ 404 오류 발생 시**
1. **Branch와 Folder 설정 재확인**
2. `docs/whitepaper/index.html` 파일 존재 확인
3. 5-10분 추가 대기 후 재접속

### **❌ CSS/JS 로딩 실패 시**
1. 상대 경로 확인 (`./styles/` → `styles/`)
2. 파일명 대소문자 정확히 확인
3. 브라우저 캐시 삭제 후 재접속

### **❌ 이미지 표시 안됨**
1. 이미지 파일 경로 확인
2. GitHub에 이미지 파일 업로드 확인
3. 파일 크기 제한 (25MB 이하) 확인
