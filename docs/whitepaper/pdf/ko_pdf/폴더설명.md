# 📁 ko_pdf 폴더 - AKASSECT 한국어 백서 컬렉션

## 🎯 폴더 목적
이 폴더는 AKASSECT 공식 백서의 한국어 버전을 버전별로 정리하여 보관하는 공간입니다.

## 📄 포함될 문서들

### 🟢 정식 버전 (Release Version)
- **파일명**: `AKASSECT-공식백서-v1.0-정식버전-한국어-YYYY-MM-DD.pdf`
- **용도**: 
  - 공식 배포용
  - 투자자 제출용
  - 파트너사 공유용
  - 공개 다운로드용
- **특징**:
  - 녹색 "정식버전" 배지
  - 완성된 모든 디자인 요소
  - 최종 검토 완료된 내용

### 🟡 임시 버전 (Draft Version)  
- **파일명**: `AKASSECT-공식백서-v1.0-임시버전-한국어-YYYY-MM-DD.pdf`
- **용도**:
  - 내부 검토용
  - 피드백 수집용
  - 수정 전 임시 배포용
  - 개발팀 공유용
- **특징**:
  - 주황색 "임시버전" 배지
  - 수정 가능 상태 표시
  - 검토 진행 중임을 명시

## 🚀 생성 방법

### 자동 생성 (권장)
```bash
cd whitepaper-sections
./export-ko-pdf.sh
```

### npm 스크립트 사용
```bash
cd whitepaper-sections
npm run export-ko
```

### 수동 생성
```bash
cd whitepaper-sections
npm install
python3 -m http.server 8082 &
node export-to-ko-pdf.js
```
