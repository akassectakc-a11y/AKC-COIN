# 🦊 MetaMask에 AKC 토큰 추가하기

## 🔗 **원클릭 추가 링크**

아래 링크를 클릭하면 MetaMask에 AKC 토큰이 자동으로 추가됩니다:

```
https://metamask.github.io/Add-Token/?address=0x02D8b729885290a3CA724F3Df5793b74Ff226A17&symbol=AKC&decimals=18&image=https://github.com/akassectakc-a11y/AKC-COIN/raw/main/docs/logo/png/AKC-Icon-256.png
```

## 📱 **수동 추가 방법**

### **1. MetaMask 열기**
- 브라우저 확장 프로그램 또는 모바일 앱

### **2. 토큰 추가**
1. **자산** 탭에서 **토큰 가져오기** 클릭
2. **사용자 정의 토큰** 선택
3. 다음 정보 입력:

```
토큰 컨트랙트 주소: 0x02D8b729885290a3CA724F3Df5793b74Ff226A17
토큰 기호: AKC
소수 자릿수: 18
```

### **3. 토큰 로고**
- MetaMask가 자동으로 로고를 가져옵니다
- 또는 수동으로 이미지 URL 추가:
```
https://github.com/akassectakc-a11y/AKC-COIN/raw/main/docs/logo/png/AKC-Icon-256.png
```

## 📊 **토큰 정보 확인**

추가 후 다음 정보가 표시되어야 합니다:
- **이름**: AKASSECT
- **기호**: AKC
- **네트워크**: BSC (Binance Smart Chain)
- **잔액**: 보유한 AKC 토큰 수량

## 🔍 **문제 해결**

### **토큰이 표시되지 않는 경우:**
1. **네트워크 확인**: BSC 메인넷인지 확인
2. **주소 확인**: 컨트랙트 주소가 정확한지 확인
3. **새로고침**: 지갑 새로고침 또는 재시작
4. **잔액 확인**: 실제로 토큰을 보유하고 있는지 확인

### **로고가 표시되지 않는 경우:**
- 시간이 지나면 자동으로 표시됩니다
- 또는 토큰리스트에 등록 필요
